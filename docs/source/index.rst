PyStatPower documentation
=========================

Introduction
------------

PyStatPower is a python package for power analysis in statistics. It can be used to:

- Estimate the sample size
- Calculate the power of a statistical test
- Back-calculate the effect size

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the package.
